Real Estate Data Analysis
Overview

This project analyzes real estate data to identify trends, patterns, and insights using Python and Jupyter Notebook.

Dataset

The dataset used in this project is from Kaggle:

https://www.kaggle.com/datasets/ahmedshahriarsakib/usa-real-estate-dataset

Due to GitHub file size limits, the dataset is not included in this repository.

Setup Instructions

Download the dataset from the link above

Place the file in a root directory

Open the Jupyter Notebook

Run all cells

Notes

Make sure the dataset file path matches what is used in the notebook.
